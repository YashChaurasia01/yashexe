Lerp & Duration

Events

virtualScroll &
scroll Events

virtualScroll intercepts wheel/touch events before Lenis processes them. Return false to block smooth scroll. Mutate e.deltaY to change speed.

virtual-scroll Event Log

Raw deltaX / deltaY from every wheel/touch event Lenis receives.

waiting for scroll...
lenis.on('virtual-scroll', ({deltaX, deltaY, event}) => {
  // raw delta before lerp
});

virtualScroll Option

Half speed: (e) => { e.deltaY /= 2 }
Block on Shift: ({event}) => !event.shiftKey
Block scroll: return false
These are set at init time in the config object.

virtualScroll: (e) => {
  // e.deltaY /= 2 → half speed
  // return false → block
  return true; // allow
}
Option

Nested Scroll & Prevent

When a scrollable element is inside your Lenis page, use data-lenis-prevent or the prevent option so Lenis doesn't intercept native scroll inside it.

With data-lenis-prevent

This box uses data-lenis-prevent. Scroll inside here — it scrolls natively, not smoothly. The outer page won't move when your cursor is inside.

Lenis checks DOM traversal on each wheel event and detects this attribute, then hands control back to the browser for this element.

⚠️ Use allowNestedScroll:true as an alternative — it auto-detects all scrollable children but has a performance cost since it walks the DOM tree on every scroll.

For modals, sidebars, or any nested scroll container, this is the recommended approach.

Even more content here so you can really test it.

Without data-lenis-prevent

This box has no data-lenis-prevent. Scroll inside here — Lenis intercepts the wheel event and the outer page scrolls instead of this inner box.

This is the default behaviour. The inner overflow content cannot be scrolled because Lenis claims the event.

To fix: add data-lenis-prevent to this element, or use the prevent:(node)=>... option in the Lenis constructor.

Other attributes available: data-lenis-prevent-wheel, data-lenis-prevent-touch, data-lenis-prevent-vertical, data-lenis-prevent-horizontal.

All Options Reference

new Lenis({
  autoRaf: false,  // false = you drive raf manually
  autoResize: true,  // ResizeObserver auto-updates
  autoToggle: false,  // start/stop based on overflow CSS
  lerp: 0.1,   // 0.01 buttery → 0.5 snappy
  duration: 1.2,  // replaces lerp if set
  easing: t => Math.min(1, 1.001 - 2**(-10*t)),
  orientation: 'vertical', // 'horizontal' for side-scroll
  gestureOrientation: 'vertical',
  smoothWheel: true,
  syncTouch: false,  // mimic touch on desktop
  syncTouchLerp: 0.075,
  touchInertiaExponent: 1.7,
  touchMultiplier: 1,
  wheelMultiplier: 1,
  infinite: false,  // loop scroll
  overscroll: true,  // rubber-band end zones
  anchors: true,   // smooth anchor links
  stopInertiaOnNavigate: false,
  allowNestedScroll: false,
  naiveDimensions: false,  // ⚠️ perf cost
  prevent: (node) => node.id === 'modal',
  virtualScroll: (e) => { /* intercept */ },
  wrapper: window,   // scroll container
  content: document.documentElement,
});
