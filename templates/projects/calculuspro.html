Overview

Compute fast.
Solve smarter.

CalculusPro is an all-in-one computation workspace built for students and professionals who outgrow a stock calculator app. It bundles a full scientific engine — trigonometry, matrices, statistics, derivatives, integrals, and financial functions — into a single, fast, distraction-free interface.

Beyond raw computation, CalculusPro layers in a live currency dashboard covering 150+ currencies, an interactive 2D/3D graphing engine for parametric and implicit equations, and a persistent calculation history so every past result is one tap away from being reused.

Process

How we made it

01 Engine Architecture

Designed a modular calculation core covering scientific, matrix, statistical, and calculus operations, structured so each mode plugs into a shared expression parser.

02 Live Data Integration

Wired up a real-time currency exchange feed spanning 150+ currencies with auto-refresh, and cached rates locally to keep the dashboard responsive offline.

03 Graphing Engine

Built an interactive 2D/3D plotting surface supporting parametric, implicit, and ML-driven equations, with smooth pan/zoom and live coefficient scrubbing.

04 History & Refinement

Added a persistent calculation log with one-tap reuse, then refined keypad ergonomics and layout across desktop and mobile after rounds of real-use testing.
