      01

      DeepSearch — C.B.I.R.

      DeepSearch is a Content-Based Image Retrieval (CBIR) web application that enables users to find visually similar images by uploading an image, delivering an experience inspired by Google Lens on a domain-specific dataset using deep learning-based feature extraction and similarity search.

      I designed and developed the complete frontend using HTML5, CSS3, JavaScript, Bootstrap, and integrated it with Flask for seamless communication with the backend. The CBIR model and backend were developed by Utkarsh Saxena.

      Technologies: HTML5, CSS3, JavaScript, Bootstrap, Flask, Python, OpenCV, TensorFlow/PyTorch, CNN, CBIR, Feature Embeddings, Cosine Similarity.

      HTML CSS FLASK JS Python opencv bootstrap tensorflow

      02

      CropCure — Plant Disease Detection & Cure.

      CropCure is a machine learning-based plant disease detection web application that identifies diseases from uploaded leaf images and provides treatment recommendations to help farmers take timely action.

      I designed and developed the complete frontend using HTML5, CSS3, JavaScript, Bootstrap, and integrated it with Flask to communicate with the machine learning model.

      Technologies Used: HTML5, CSS3, JavaScript, Bootstrap, Flask, Python, TensorFlow/PyTorch, OpenCV, CNN, Computer Vision, Machine Learning.

      HTML CSS JS tensorflow Python FLASK Machine Learning Deep learning Computer Vision

      03

      CalculusPro — Where Mathematics Meets Innovation.

      CalculusPro is a comprehensive mathematics and scientific computation platform that combines a scientific calculator, real-time currency converter, 2D & 3D graph visualization, and calculation history into a single modern web application. Designed with a focus on usability, it provides an intuitive interface for solving mathematical problems and visualizing functions.

      I designed and developed the complete application using HTML5, CSS3, JavaScript, implementing responsive layouts, interactive graphing, and a clean, user-friendly interface.

      Technologies: HTML5, CSS3, JavaScript, Mathematical Computation, Graph Visualization, Scientific Calculator, Currency Exchange API, Responsive Design, UI/UX.

      HTML CSS JS Doc Guide 2D|3D Graph Visualization Use History Live Currency Exchange Monitoring Usage Manual

      04

      Parts Drive India — Mitsubishi Auto Parts Inventory.

      Parts Drive India is a modern automotive platform designed to help users discover genuine Mitsubishi spare parts, reducing the risk of counterfeit products and improving trust in the purchasing process.

      The platform features seamless WhatsApp API integration for direct customer-to-business communication, enabling users to quickly inquire about parts and receive personalized assistance without creating an account.

      I designed and developed the complete website with a focus on responsive design, performance, accessibility, and cross-device compatibility, ensuring a smooth experience across desktops, tablets, and mobile devices.

      Technologies: HTML5, CSS3, JavaScript, WhatsApp API, Responsive Design, UI/UX, Performance Optimization.
